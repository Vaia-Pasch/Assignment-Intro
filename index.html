What is Frontend?

There are 3 parts

HTML

Markup Language

CSS

Cascading Style Sheets

JAVASCRIPT

Programming language

HTML a.k.a. The builder

Can we fix it? Yes we can!

HTML is a HyperText Markup Language and is our structure. It defines all the content, the images and the links. HTML is on almost every single page on the internet, so keep it close to you, don't let it out of your sight.

CSS a.k.a. The artist

Building without Art is just...EH!

Creative spirit ON! CSS is responsible for colors, fonts and the positioning of our content. CSS is very clingy with HTML, like very very very clingy.

JAVASCRIPT a.k.a. The wizard

Call it magic, call it JS

JavaScript is a programming language used in web development. JS code can be inserted anywhere within the HTML of a webpage, talking about another clingy fella. To put it simple, js is interacting with our user or in two words: Wingardium leviosa.
